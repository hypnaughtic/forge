# Nuxt Fullstack Template (STUB)

> **Status:** Stub template -- not yet scaffolded. This document describes what a full implementation would provide.

## Overview

A fullstack application powered by Nuxt 3 with Nitro server engine and Drizzle ORM for type-safe database access. Combines SSR, API routes, and database in a single deployable unit.

## What a Full Scaffold Would Provide

- **Nuxt 3** with auto-imports, file-based routing, and hybrid rendering (SSR/SSG/SPA)
- **Nitro server engine** with API routes, middleware, and server utilities
- **Drizzle ORM** with type-safe schema, migrations, and query builder
- **Authentication** with nuxt-auth-utils or Lucia
- **Tailwind CSS** with Nuxt module integration
- **Form handling** with server-side validation (zod)
- **SEO optimization** with useHead, meta tags, and sitemap generation
- **Error handling** with custom error pages and createError
- **Caching strategies** with Nitro route rules
- **Testing** with Vitest for unit and Playwright for E2E
- **Deployment configuration** for Vercel, Cloudflare, or Docker

## Key Technologies

| Component       | Technology          |
|----------------|---------------------|
| Framework       | Nuxt 3              |
| Server Engine   | Nitro               |
| ORM             | Drizzle             |
| Styling         | Tailwind CSS        |
| Validation      | Zod                 |
