# Vue SPA Template (STUB)

> **Status:** Stub template -- not yet scaffolded. This document describes what a full implementation would provide.

## Overview

A modern single-page application built with Vue 3 Composition API, Vite for lightning-fast development, Pinia for state management, and Tailwind CSS for utility-first styling.

## What a Full Scaffold Would Provide

- **Vue 3 with Composition API** and script setup syntax throughout
- **Vite build tooling** with HMR, optimized production builds, and environment configuration
- **Pinia stores** with typed state, getters, and actions
- **Vue Router** with lazy-loaded routes, navigation guards, and route-level code splitting
- **Tailwind CSS** with custom theme configuration and purged production builds
- **Component library** structure with base components and composables
- **API layer** with Axios or fetch wrapper, interceptors, and error handling
- **Authentication flow** with token management and protected routes
- **Form handling** with VeeValidate or FormKit
- **Internationalization** setup with vue-i18n
- **Unit testing** with Vitest and Vue Test Utils
- **E2E testing** scaffold with Playwright or Cypress
- **ESLint + Prettier** configuration tuned for Vue
