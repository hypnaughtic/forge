Project: CLI data pipeline tool for ETL with plugin architecture

Build a CLI data pipeline tool. Features: declarative YAML pipeline definitions, built-in extractors (CSV, JSON, API, DB), transformers (filter, map, aggregate, join), loaders (PostgreSQL, S3, BigQuery), plugin architecture for custom extractors/transformers/loaders, parallel execution with configurable concurrency, dry-run mode, progress reporting, error handling with dead-letter queue, pipeline versioning and migration support. Tech: Python with Click CLI, async I/O, no frontend.
